🚗 Sistema de Gestión de Peajes

Proyecto académico desarrollado como obligatorio para la materia Diseño de Aplicaciones (ORT).
Consiste en un sistema web para la gestión de vehículos, propietarios y tránsitos en puestos de peaje.

🎯 Objetivo

Modelar e implementar un sistema de peajes aplicando buenas prácticas de diseño, patrones y arquitectura MVC, cumpliendo con requerimientos funcionales y no funcionales definidos.

🧩 Funcionalidades principales
👤 Aplicación para Propietarios
Login de usuario
Visualización de:
Datos personales y estado
Saldo actual
Vehículos registrados
Historial de tránsitos
Bonificaciones asignadas
Notificaciones del sistema
Eliminación de notificaciones

🛠️ Aplicación para Administradores
Login de administrador
Emulación de tránsitos
Asignación de bonificaciones a propietarios
Cambio de estado de propietarios
Generación automática de notificaciones

🌐 Interfaz Web
SPA (Single Page Application) sin recarga de página
Persistencia de estado ante refresh
Actualización automática de datos
Manejo de todos los casos alternativos definidos
